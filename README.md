## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Swagger docs
```bash
# Documentation
$ http://localhost:3150/api/
```

## Clone repository
```bash
# clone
$ https://github.com/charlesbrendon/pizza-fresh-server
```
## Installation

```bash
# installation
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
